Seven Segment Display Converter Using Verilog HDL

This project implements a Seven Segment Display Converter in Verilog HDL. The design converts a 4-bit BCD input into the corresponding seven-segment display outputs (a–g) for displaying decimal digits 0–9. Invalid BCD inputs (10–15) are handled by turning off all segments. The project includes a testbench for functional verification and simulation in Xilinx Vivado.
